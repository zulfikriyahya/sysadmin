# Setup git

## Inisialisasi

```
git init
```

## Menambahkan seluruh isi dalam direktori project

```
git add .
```

## Commit project

```
git commit -m "Initial Project."
```

## Menginisiasi branch

```
git branch -M main
```

## Menginisiasi repository

```
git remote add origin https://github.com/<username>/<repository>.git
```

## Push project

```
git push -u origin main
```
